# 🏫 Campus Found - Lost & Found Platform

A premium, glassmorphism-inspired **Lost & Found** platform designed specifically for college campuses. Built with **FastAPI** and **Supabase**, it ensures a secure and seamless experience for students to report lost items and recover them.

---

## ✨ Features

- **🔐 Secure Authentication**: Exclusive sign-up for college students using `@*.rvitm@rvei.edu.in` email IDs.
- **🛡️ Domain Guard**: Prevents non-college emails from accessing the platform.
- **📱 Responsive UI**: A modern, dark-mode glassmorphism design that looks stunning on desktop and mobile.
- **⚡ Real-time Database**: Powered by Supabase for instant updates and reliable storage.
- **🔄 Auto-Resolve Flow**: When a "Lost" item is marked as resolved, it automatically moves to the "Found Items" board.
- **📧 Email Verification**: Integrated OTP/Link verification to ensure user authenticity.

---

## 🛠️ Technology Stack

- **Backend**: Python (FastAPI)
- **Frontend**: HTML5, Vanilla CSS (Glassmorphism), JavaScript
- **Database**: PostgreSQL (via Supabase)
- **Auth**: Supabase Auth
- **Deployment**: Vercel

## 🚀 Getting Started

### Prerequisites
- Python 3.10+
- A Supabase Project

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/lakshmipathy0905-oss/campus-lost-found.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up environment variables:
   - `SUPABASE_URL`
   - `SUPABASE_KEY`
4. Run the server:
   ```bash
   python app.py
   ```
